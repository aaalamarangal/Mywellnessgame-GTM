# 2.content — Recognising Finding Demand

This folder contains the operational layer of finding demand.
It documents the actual methods used to find founder posts in the field.

## Files

- **founder-post-sources.md** — the method Shiva uses to find founder posts worth messaging on. One method documented: Algorithm Feed → Productivity Conversation → Profile Check. This file also serves as the early spec for an AI-native software that will assist with this process.

## How this connects

The 1.context/ folder explains how to read and classify signals once found.
This 2.content/ folder documents how to find those signals in the first place.
They work together: find first (here), then read and classify (context layer).
