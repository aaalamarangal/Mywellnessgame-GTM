## Tagging system

Every entry header carries three tags:

**Tag 1 — Person type**
- `[CLIENT]` — Sujith (paying client, being coached)
- `[SELF]` — Shiva (coach running the system on themselves)
- `[TEAM]` — Vinotha (team member running the system herself)

**Tag 2 — Leverage level**
- `[HIGH]` / `[MEDIUM]` / `[LOW]`

**Tag 3 — Active game in nano constraints (Input 7)**
- `[CLARITY]` / `[ACTION]` / `[LESSON]`

Tag 3 goes inline at Input 7, not in the header.
