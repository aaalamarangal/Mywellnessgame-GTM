Parent: 3.Remembering fulfilling demand | CLAUDE.md

## What this folder is

Content entries for the fulfilling layer of Value Delivery.
The 12-input Macro → Micro → Nano template is logged here for every active delivery session.

## Files

- **delivery-log.md** — master log for all delivery sessions. One entry per session. Tagged by person type / leverage / game.
- **suggestions-log.md** — one suggestion per entry, grouped by date. Shared only when Sujith asks. Each suggestion connects to a pattern observed in delivery-log.md.

## How to add a delivery entry

Copy the 12-input template from delivery-log.md.
Set the correct person type tag and leverage tag in the header.
Set the game tag at Input 7.
Fill in all 12 inputs. Do not skip any field.

## How to add a suggestion

Copy the suggestion template from suggestions-log.md.
Link the suggestion to a specific pattern in delivery-log.md.
Mark status as "Not yet shared" until Sujith asks and it is shared.
