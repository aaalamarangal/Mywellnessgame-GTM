# Demo Posting
Parent: 3.Recognising fulfilling supply | CLAUDE.md

## What demo posting is

Demo posting is LinkedIn posts that demonstrate the coaching skill publicly.
It is part of Offer Creation OS — the fulfilling layer.

It is NOT the same as demo commenting (which is Value Creation OS).
Demo posting is public and broadcast. Demo commenting is 1:1 and private.

Both lead to the same place: a demo call.

---

## How demo posting differs from demo commenting

| | Demo commenting | Demo posting |
|---|---|---|
| OS | Value Creation | Offer Creation |
| Mode | 1:1, private | Public, broadcast |
| Trigger | Founder signal | Shiva's insight |
| Audience | One specific founder | Anyone on LinkedIn |
| Goal | Complete 3 jobs with that founder | Attract founders who self-select |

---

## What a demo post does

A demo post shows the skill in action — not explains it.

It does one of three things:
1. Makes the game visible (shows what Clarity / Action / Lesson looks like in practice)
2. Shows the regulation system working (real example, specific result)
3. Names the gap that founders don't have words for yet

A demo post is not a pitch. It is a demonstration.
The founder who reads it should think: "That is exactly what I am experiencing."

---

## Principles

1. **Show, don't explain.** Use a real situation. Not "here is how the system works."
2. **One game per post.** Clarity OR Action OR Lesson. Not all three.
3. **Exact words.** When quoting founders, use their exact words. Never paraphrase.
4. **Under 200 words.** Short posts get read. Long posts get scrolled.
5. **No call to action.** Let the post do the work. Founders who fit will reach out.
