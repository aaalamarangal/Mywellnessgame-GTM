## > INSPIRATION Workflow

Triggered when Shiva shares an external quote, insight, or idea (GTM post, podcast, field observation).

Steps:
1. Identify source type: GTM post / video-podcast / field observation
2. Extract key insight — exact language, mapped to 3 games or Leverage Stack
3. Write 2-3 post angles (marked "when ready")
4. Add row to Post Arc Tracker (Stage: Captured)
5. Add full entry to the correct section in post-inspiration.md (Status: Captured)
6. Confirm in chat: "Logged. Source: [name]. Stage: Captured."

Note: Value Creation and Value Delivery sources are added manually by Shiva when post-worthy.
Use the same format: Status, exact quote, key insights, post angles.

---

## > POST Workflow — Post Arc Connection

Before writing any post:
1. Read the Post Arc Tracker in post-inspiration.md
2. Surface all entries at Stage: Captured or Angle ready
3. Shiva picks which one to develop
4. Develop the angle → write the draft (under 100 words)
5. When published:
   - Update Post Arc Tracker row: Stage → Published, add post link
   - Update full entry: Status → Published, add post link
   - Add entry to posts-log.md immediately
